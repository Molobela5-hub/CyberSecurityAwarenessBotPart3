Cybersecurity Awareness Bot
Project Overview
This is a C# console-based chatbot developed for cybersecurity awareness. It helps users learn about password safety, phishing, suspicious links, and safe browsing.

Features
Voice greeting using WAV audio
ASCII art title screen
Personalised user interaction
Basic cybersecurity responses
Input validation
Coloured console interface
Typing effect
Structured code using classes and methods
GitHub Actions CI workflow
How to Run
Open the project in Visual Studio.
Ensure the file Assets/greeting.wav exists.
Build and run the application.
