# CyberSecurity Awareness Bot - Part 2

## Project Overview

The CyberSecurity Awareness Bot is a C# WPF-based chatbot designed to educate users about cybersecurity awareness and online safety. The application provides interactive conversations, cybersecurity guidance, sentiment detection, memory recall, and cybersecurity challenges.

## Features

### User Interface

* WPF graphical user interface
* Chat-style interaction
* User-friendly design

### Cybersecurity Awareness

* Password security advice
* Phishing awareness
* Scam prevention
* Safe browsing guidance
* Cybersecurity tips

### Sentiment Detection

The chatbot can detect emotions such as:

* Worried
* Confused
* Frustrated
* Happy
* Curious

The bot adapts its responses based on the user's sentiment.

### Memory and Recall

The chatbot can:

* Remember the user's name
* Recall the user's name when asked
* Remember the user's favourite cybersecurity topic
* Provide personalised cybersecurity tips

### Interactive Challenges

Users can test their cybersecurity knowledge through:

* Phishing detection challenges
* Password security challenges
* Public Wi-Fi safety challenges

### Additional Features

* Voice greeting using WAV audio
* ASCII art asset included from Part 1
* Delegate implementation
* Dictionary and List collections
* Randomized responses
* Error handling
* Conversation flow support

## Technologies Used

* C#
* .NET
* WPF (Windows Presentation Foundation)
* Visual Studio

## Project Structure

### Services

* ResponseService
* SentimentService
* MemoryService
* ChallengeService
* AudioPlayer

### Assets

* Greetings.wav
* ascii_art.txt

## How to Run

1. Open the solution in Visual Studio.
2. Ensure the Assets folder contains Greetings.wav.
3. Build the solution.
4. Run the application.
5. Interact with the chatbot using the graphical interface.

## Example Commands

* My name is John
* What's my name?
* My favourite topic is phishing
* Tell me a tip
* Tell me about passwords
* Tell me about phishing
* Start challenge
* I am worried about phishing
